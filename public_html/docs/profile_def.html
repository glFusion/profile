Custom Profile Field Definitions

Field Definitions
Field Name This is the name of the custom field used in the database. It must be unique.
Text Prompt This is the text that will be displayed next to the fields in forms and display screens. For example, a custom field named "birthdate" might have a prompt of "Birth Date".
Field Type Select the type of field to be used. This selection will determin which of the remaining options are displayed.
Length (Type=text) These values determind the display length and actual maximum length of entries that can be made.
Default Value (Type=text, radio, checkbox) Enter the default value for this field. For text fields, this is a simple string. For radio buttons, enter the value corresponding to the option to be selected as default. For checboxes, enter either 1 or 0.
Field Mask (TESTING) (Type=text) Enter the mask string to be used with iMask for this field.
  • "9" represents a digit.
  • "A" represents a letter.
  • "X" represents an alphanumeric character.
Show Time? (Type=date) Check this box if the time selector is to be available on the datepicker.
12 or 24-hour format? (Type=date) Select whether times will be entered in 12-hour or 24-hour format.
Format (Type=date) Enter the format string to be used for the date/time. Refer to the PHP date formats for more information.
Value (Type=radio, dropdown) This is a string which indicates the possible values for dropdown selections and radio buttons. The format is "value:text;value:text;..." For example, to have two radio buttons labeled "Red" and "Blue", which return values of "R" and "B" respectively, you would enter "R:Red;B:Blue" as the value.
Readonly Checking this box indicates that regular users cannot change the values.
Enabled Check this box to enable the field, uncheck to disable. This allows the administrator to prevent some fields from being used without deleting them. (Deleting a custom field also deletes all user data associated with it).
Required Check this box to make the field required. This also implies that the field will be displayed on the user registration page.
Enter at Registration? Check this box to have the field appear on the user signup page. This is implied if 'required' is selected.
Position After Select the existing custom field after which this field will appear. New fields appear at the bottom of the list by default. You may change the field order at any time.
