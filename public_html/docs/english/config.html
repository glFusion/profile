Profile Plugin

The Profile Plugin allows you to create custom profile fields for your user accounts.

This version requires GLFusion 1.2.0 or higher.

Profile Plugin Configuration Options

Main Profile Settings
Option Description
Display Empty Values? Select whether empty values will be displayed on the profile views. If this is "False", then fields that have no value will not be displayed at all.
Expiration Grace Period If the system expiration date is used, this value will indicate the number of days after expiration when the date will be highlighted as being expired. The highlight style is defined in <plugin_dir>/css/styles.css.
Date Format Select the default format used for date fields. This may be overridden for each field.
Permissions These are the default permissions that will be applied to newly-created fields. They may be overridden during or after creation.
Include Admin in lists? Select whether to include the Administrator account, UID 2, in profile lists. This should typically be "No"
Allow PDF export of lists? Select whether regular users who are allowed to view lists are also allowed to export lists to PDF files. This is separate from the "profile.export" permissions and is global to all lists.
