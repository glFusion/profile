Custom Profile List Definitions

Field Definitions
List ID This is the unique ID of the custom list. This string is used to create the url to view the list.
Title This is the title that will be displayed.
Viewable By Select the user group that is allowed to view this list.

"All Users" includes both logged-in site members and anonymous users. "Logged-In Users" includes all logged-in site members. Other groups can be defined by the administrator.

Group to Include Select the group that will be included in this list.

"All Users" includes both logged-in site members and anonymous users. "Logged-In Users" includes all logged-in site members. Other groups can be defined by the administrator.

Order Set the order in which this list will appear in the menu. The order can also be changed by clickin the blue arrows on the main list screen.
